  - PlaceholderAPI Support
  - Automatic Pack Generation
  - Animated Components
  - Text Generation Speed
  - Text Generation Actions
  - Pre & Post Actions
  - Switch & Select Actions
  - Select & Unselected Component Rendering
  - Ability to implement your own font
  - Public API
