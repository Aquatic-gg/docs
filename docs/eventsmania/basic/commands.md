# COMMANDS
Base command:
- event

:::tip

All commands require `eventsmania.admin` to use.

:::

## KEY

  **`<event>`**
    - The name of the event. (ex. "blockbreak")

## MORE COMMANDS

  `event start <event>`
    - Start an event forcefully.

  `event stop`
    - Stop the running event.

  `event reload`
    - Reload the plugin.
