# CLICK ACTIONS

Click actions are when players click on the npc.

## TYPES

What type of hit should we register.

Options: `LEFT`, `RIGHT`, `SHIFT_LEFT`, `SHIFT_RIGHT

```yml
click-actions:
  - types:
      - LEFT
      - SHIFT_LEFT
    actions:
      ... # Look at actions page for more info
```
