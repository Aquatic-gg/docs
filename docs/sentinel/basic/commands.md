# COMMANDS
Base command:
- sentinel

:::tip

All commands require `sentinel.admin` to use.

:::

## KEY

  **`<npc>`**
    - The name of the npc file. (ex. "discord")

## MORE COMMANDS

  `sentinel spawn <npc>`
    - Spawn an npc at your location.

  `sentinel reload`
    - Reload the plugin.
